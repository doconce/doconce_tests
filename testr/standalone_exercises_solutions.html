Problem 2: Flip a Coin

Answer. If the random.random() function returns a number \( < 1/2 \), let it be head, otherwise tail. Repeat this \( N \) number of times.

a) Solution. 

import sys, random
N = int(sys.argv[1])
heads = 0
for i in range(N):
    r = random.random()
    if r <= 0.5:
        heads += 1
print('Flipping a coin %d times gave %d heads' % (N, heads))
